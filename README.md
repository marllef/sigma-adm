## SigmaADM
Um software de gestão de recursos e encomendas desenvolvido para facilitar a administração de pequenas organizações.

## Tecnologias
Algumas das tecnologias utilizadas neste projeto são:

* NextJS
* TypeScript
* Prisma ORM
* PostgreSQL
